# Feature Guide
!!! note ""
	__Here is the Feature Guide of wavlink, where you can view all the features of the wavlink device.__

	__Their functional interfaces are not completely consistent, so we provide classified viewing.__

	__Some special functions are only available on special devices, such as ADGuard, USB Management, etc. These functions are only available on some routers, and other products such as indoor repeaters and outdoor APs are not currently applicable.__

	__Therefore, you can view the information you need based on product categories.__
